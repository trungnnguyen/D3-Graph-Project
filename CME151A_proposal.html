CME 151A Final Project Proposal

Working Title: Graph Visualization Project

Group Members: Vishal Subbiah and Alex Infanger

We plan to create a web app which accepts an adjacency, incidence or Laplacian matrix data file in any of the standard formats (CSR,CSC,COO, dense) and then outputs a graph, where edge weights are translated to forces. (If the user does not provide edge weights then they are all set to 1). This project, in effect, will be a generalization and modfiication to Mike Bonstock’s ``Force Directed Graph’’ for applications in engineering.

We expect to use the d3.forces layout and JSON data format. Our most involved tasks include: (1) writing code to convert any matrix format into JSON and (2) making a beautiful matrix data upload web app.

One question we might hope to answer with this data visualization is whether or not the structure of a matrix can be identified with the structure of the graph upon reaching equilibrium with respect to the edge-forces. We hypothesize that the structure imposed by positing a force law for the edges overshadows any structure that might be inherent in the matrices and the systems they represent (for example, physical systems in the case of PDEs). One way to test this would be to see if similar structure arises from randomly generated matrices.
